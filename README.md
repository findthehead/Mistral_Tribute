# Mistral_Tribute 
Mistral is very generous with their free API so here you can use it too with Mistral CLI and Mistral GUI scripts.

----


- Make an account with [*Mistral Website*](https://console.mistral.ai/)
- Create API key
----

### If you want to use CLI app, run this below command.
```python
cd Mistral_Tribute
#Create an API key and place it to .env file
pip install -r requirements.txt
python3 mistral_cli.py
```
----
### If you want to use GUI app, run this below command.

```python
cd Mistral_Tribute
#Create an API key and place it to .env file
pip install -r GUI_requirements.txt
streamlit run mistral_gui.py
```
